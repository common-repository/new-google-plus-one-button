== Description ==
 
The Google Plus One Plugin adds to every post a Google Plus One Button.
 
== Installation ==

1. Upload the Plugin directory to the "/wp-content/plugins/" wordpress directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Optional: Change the size of Google Plus One Button in your admin panel.
